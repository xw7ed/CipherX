# CipherX - Encryption/Decryption Tool

## Requirements
- Python 3.6+
- tkinter (sudo apt install python3-tk)
- pycryptodome (pip3 install pycryptodome)

## Usage
1. Run the script: `python3 CipherX.py`
2. Choose the cipher type (Caesar, XOR, Vigen√®re, AES).
3. Enter the text and key/shift.
